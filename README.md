# TypeScript config

> Base TypeScript config for composite projects.

## Usage

```bash
yarn add -D @anireact/typescript-config
```

```bash
npm install --save-dev @anireact/typescript-config
```

```jsonc
// tsconfig.json
{
    "extends": "@anireact/typescript-config"
}
```
